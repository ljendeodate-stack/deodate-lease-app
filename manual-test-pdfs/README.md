# Manual Test PDF Fixtures

Prepared: 2026-03-30

This folder contains sample PDF lease documents for manual intake testing through the current `Scan Lease` upload path.

The fixture set is split across two behaviors already present in the app:

- Structured PDF parser cases: the PDF contains a digitally generated table that the app can parse directly.
- OCR-first narrative cases: the PDF contains lease prose or non-standard layouts intended for the OCR extraction path and human review flow.
